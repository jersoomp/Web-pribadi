# Situs Web Pribadi

Ini adalah situs web pribadi Jeremy Sompie, seorang mahasiswa dari Universitas SamRatulangi Manado.

## Deskripsi

Situs web pribadi sederhana untuk berbagi pengalaman dan minat dalam gaming.

## Halaman

- **Home** (`index.html`): Pengantar dan navigasi ke halaman lainnya.
- **Galeri** (`Gallery.html`): Koleksi foto.
- **Blog** (`Blog.html`): Artikel tentang AI dan gaming.
- **Kontak** (`Contact.html`): Informasi kontak.


## Teknologi yang Digunakan

- HTML

Galeri menyertakan berbagai tangkapan layar dan foto yang disimpan di direktori root.
